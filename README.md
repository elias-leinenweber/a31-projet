Instructions
------------
### Installation
1. Télécharger l'archive `connect4.jar`.
2. Dans une fenêtre de terminal, entrer la commande `java -jar connect4.jar`
   dans le répertoire où l'archive JAR est située.

(La version Java 14 est recommandée.)
